# Development

Install dependencies:

```bash
npm install
```

Start a dev server locally:

```bash
npm run serve
```

This will hot reload updates, and output those changes to the `/docs` directory.

# Deploy

Commit and push.

```bash
git add .
git commit -m 'message'
git push
```
